9. –°–æ–∑–¥–∞–π—Ç–µ –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä `@memoize`, –∫–æ—Ç–æ—Ä—ã–π –∫—ç—à–∏—Ä—É–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤. –ï—Å–ª–∏ —Ñ—É–Ω–∫—Ü–∏—è –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è —Å —Ç–µ–º–∏ –∂–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∞–º–∏, —á—Ç–æ –∏ –≤ –ø—Ä–æ—à–ª—ã–π —Ä–∞–∑, –æ–Ω–∞ –¥–æ–ª–∂–Ω–∞ –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∏–∑ –∫—ç—à–∞, –∞ –Ω–µ –≤—ã—á–∏—Å–ª—è—Ç—å –µ–≥–æ –∑–∞–Ω–æ–≤–æ.
    
    –¢–∞–∫–∂–µ —Å–æ–∑–¥–∞–π—Ç–µ –¥–≤–∞ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä–∞:
    
    - `@slowit(n)` - –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä —Å –ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º–∏, –∫–æ—Ç–æ—Ä—ã–µ –∑–∞–º–µ–¥–ª—è–µ—Ç —Ä–∞–±–æ—Ç—É —Ñ—É–Ω–∫—Ü–∏–∏ –Ω–∞ `n` —Å–µ–∫—É–Ω–¥. –ë–µ–∑ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä –∑–∞–º–µ–¥–ª—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é –Ω–∞ 1 —Å–µ–∫—É–Ω–¥—É. –í –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `time.sleep(n)`.
    - `@timeit` - –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –≤—ã–≤–æ–¥–∏—Ç –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏.
    
    ```python
    # –ë–µ–∑ –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏—è –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –ø—Ä–∏ –∫–∞–∂–¥–æ–º –≤—ã–∑–æ–≤–µ –Ω–µ –º–µ–Ω–µ–µ 2 —Å–µ–∫—É–Ω–¥.
    @timeit
    @slowit(2)
    def product(n):
        return reduce(lambda x, y: x * y, range(1, n+1)) if n > 0 else None
    
    product(10)
    product(10)
    
    # –° –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏–µ–º –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –ø—Ä–∏ –ø–µ—Ä–≤–æ–º –≤—ã–∑–æ–≤–µ –Ω–µ –º–µ–Ω–µ–µ 2 —Å–µ–∫—É–Ω–¥, –ø—Ä–∏ –≤—Ç–æ—Ä–æ–º –≤—ã–∑–æ–≤–µ –ø–æ—á—Ç–∏ –º–≥–Ω–æ–≤–µ–Ω–Ω–æ.
    @timeit
    @memoize
    @slowit(2)
    def product(n):
        return reduce(lambda x, y: x * y, range(1, n+1)) if n > 0 else None
    
    product(10)
    product(10)
    ```
